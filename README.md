Welcome!

to run this program in your terminal you must make sure you have the most recent release of Python
https://www.python.org/downloads/

check if you already have Python run the command 'python --version' in terminal 

enter cd to find out what folder / directory you're currently in
navigate to the correct one, example: 
if downloaded program to Downloads on Windows enter command
'cd Downloads'

----------------------------------------------
 
 You will also need to have requirements.txt installed using 'pip install -r requirements.txt' command in terminal
 this txt file contains packages/libraries that need to be installed for this program - ie. tabulate

 * you may need to make sure that pip is newest version - run '-m pip install --upgrade pip' in terminal if required
